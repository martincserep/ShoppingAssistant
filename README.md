# ***Shopping Assistant***
> This is a shopping list and shopping cart counter app.


## Installation

  1. Clone the repository
  ```
  git clone https://github.com/martincserep/ShoppingAssistant.git
  ```
  2. Go to the created folder
  ```
  cd ShoppingAssistant
  ```


## Development setup

  1. Install
  ```
  npm install
  ```
  2. Run
  ```
  ionic serve
  ```
